# Análise Estatística com Python 📊

Este projeto faz parte dos meus estudos no curso **prograMaria - Meus Primeiros Passos em Python**. 

## 🎯 Objetivo
O script tem como objetivo identificar **outliers** (valores discrepantes) em uma lista de dados (neste caso, idades), utilizando conceitos de:
- Média e Desvio Padrão.
- Identificação de limites (Superior e Inferior).
- Visualização de dados com Boxplot.

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- **Numpy**: Para cálculos matemáticos.
- **Matplotlib**: Para geração de gráficos.

## 📈 Exemplo de Visualização
O uso do Boxplot permite identificar visualmente que o valor `400` foge completamente do padrão da amostra, sendo considerado um erro de dado ou um caso extremo.
